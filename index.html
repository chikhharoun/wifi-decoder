1
إنشاء حساب GitHub
إذا ليس عندك حساب، سجّل مجاناً من الرابط أدناه.
🔗 إنشاء حساب GitHub
2
إنشاء مستودع (Repository) جديد
بعد تسجيل الدخول:
  • اضغط + في الأعلى ثم New repository
  • اسم المستودع: wifi-decoder
  • اجعله Public
  • اضغط Create repository
3
رفع الملفات
داخل المستودع الجديد:
  • اضغط Add file ثم Upload files
  • ارفع: index.html manifest.json sw.js icon-192.png icon-512.png
  • اضغط Commit changes
4
تفعيل GitHub Pages (السيرفر المجاني)
  • اذهب إلى Settings في المستودع
  • من القائمة اليسرى اضغط Pages
  • تحت Source اختر Deploy from a branch
  • اختر Branch: main والمجلد / (root)
  • اضغط Save
بعد دقيقة يكون رابط تطبيقك:
https://[اسم-المستخدم].github.io/wifi-decoder
5
تثبيت التطبيق على الهاتف
  • افتح الرابط في Chrome على الأندرويد
  • ستظهر رسالة "تثبيت التطبيق" تلقائياً
  • اضغط تثبيت ✅
  • التطبيق يشتغل بدون إنترنت بعد التثبيت 📴
