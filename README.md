# 📲 WiFi Decoder — تطبيق فك تشفير الواي فاي

## خطوات الرفع على GitHub Pages (مجاناً)

### 1. إنشاء حساب GitHub
- اذهب إلى https://github.com
- سجّل حساباً مجانياً

### 2. إنشاء مستودع جديد
- اضغط على زر **New** أو **+**
- اسم المستودع: `wifi-decoder`
- اجعله **Public**
- اضغط **Create repository**

### 3. رفع الملفات
ارفع هذه الملفات الأربعة:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

اضغط **Add file** ثم **Upload files**

### 4. تفعيل GitHub Pages
- اذهب إلى **Settings** في المستودع
- اضغط على **Pages** من القائمة اليسرى
- تحت **Source** اختر **Deploy from a branch**
- اختر branch: **main** والمجلد **/ (root)**
- اضغط **Save**

### 5. انتظر دقيقة واحدة
سيكون التطبيق متاحاً على:
```
https://[اسم-المستخدم].github.io/wifi-decoder
```

### 6. تثبيت التطبيق على الهاتف
- افتح الرابط في **Chrome** على الأندرويد
- ستظهر رسالة **"تثبيت التطبيق"** تلقائياً
- اضغط **تثبيت** ✅
- سيظهر التطبيق على شاشتك الرئيسية!

---
التطبيق يعمل **بدون إنترنت** بعد التثبيت.
